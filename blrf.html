BLRF

An R package that applies the bag of little bootstraps to random forests.

  • 2020-3-20
  • Team Members: Yujia Lu, Yating Ge, Yuqing Yang

We write an R package called BLRF. This library combines bag of little bootstraps, also known as BLB, to replace the process of ordinary bootstrapping in the standard Random Forest algorithm in order to gain more robustness in prediction for random forest classification as well as regression. Users could call the function `blrf` to train a BLRF model, and get the prediction results for classification or regression, as well as the confidence interval of estimators. Bag of little bootstraps (BLB)

BLB is a procedure that incorporates features of both the bootstrap and subsampling to yield a robust, computationally efficient means of assessing the quality of estimators. BLB functions by combining the results of multiple subsets, and the procedure is shown in the plot above.

  • First, sample without replacement to split the training dataset into s subsample, with each subsample having a size of b. To determine b, one needs to pick a value of γ ([0,1]) and compute the value of b = n^ γ.
  • For each subsample, bootstrap it from size b to size n, the size of the original training dataset, and compute the statistics accordingly. Parallel computation could be used in this step to improve the speed.
  • Combine the results of statistics from each bootstrap subsample. One could assess the quality of estimators by using, for example, the confidence interval.

Bag of little random forests (BLRF)

BLRF is a machine learning algorithm that developed based on the algorithm of random forests. Instead of using bootstrap aggregation, the “BLB aggregation” is used to build multiple forests each built from many BLB decision trees. The pseudo-code for BLRF is shown in the plot below.
